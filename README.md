## Setting up fonts
```bash
brew tap homebrew/cask-fonts
brew search font- | grep jetbrains
brew install --cask font-jetbrains-mono-nerd-font
```
then, set the font in iTerm at Settings->Fonts->Profile->Text->Font

## Dependencies
```bash
brew install fd
```
